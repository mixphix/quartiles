# quartiles

An Apple News Quartiles solver written in Haskell.

Uses a shortened (< 16 characters) version of [this word list](https://github.com/dwyl/english-words) that I trim and expand (if necessary) with each new puzzle.

To run the solver, make sure GHC 9.6 or newer, and `cabal`, are installed and in your `PATH`. Then simply execute `cabal run` from within the project directory.
